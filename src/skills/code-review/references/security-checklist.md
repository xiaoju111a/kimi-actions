# Security Review Checklist

## 输入验证
- [ ] 所有用户输入都经过验证
- [ ] 使用白名单而非黑名单
- [ ] 验证数据类型、长度、格式
- [ ] 对特殊字符进行转义

## 注入防护
- [ ] SQL 使用参数化查询
- [ ] 命令执行避免用户输入
- [ ] XSS 输出编码
- [ ] LDAP/XML 注入防护

## 认证授权
- [ ] 密码使用安全哈希 (bcrypt/argon2)
- [ ] Session 安全生成和存储
- [ ] 敏感操作需要重新认证
- [ ] 权限检查在每个端点

## 敏感数据
- [ ] 密钥不硬编码
- [ ] 日志不包含敏感信息
- [ ] 错误信息不泄露系统细节
- [ ] 传输使用 HTTPS

## 加密
- [ ] 使用强加密算法 (AES-256, RSA-2048+)
- [ ] 避免 MD5, SHA1, DES
- [ ] 安全随机数生成
- [ ] 密钥安全存储

## 其他
- [ ] CORS 配置正确
- [ ] CSRF 防护
- [ ] 速率限制
- [ ] 安全 HTTP 头
