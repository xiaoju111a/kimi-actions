# Getting Started Guide

## Prerequisites

- GitHub repository
- Kimi API key from Moonshot AI

## Installation Steps

1. Get API key from https://platform.moonshot.cn/
2. Add to GitHub Secrets as `KIMI_API_KEY`
3. Create workflow file in `.github/workflows/`
4. Configure action inputs

## First Review

Create a PR and comment `/review` to trigger code review.
